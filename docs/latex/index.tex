\chapter{Image Filtering Library}
\hypertarget{index}{}\label{index}\index{Image Filtering Library@{Image Filtering Library}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
The \doxylink{class_image}{Image} Filtering Library provides a set of classes and functions for applying various image processing techniques to 2D images and 3D volumes. It includes support for common filters such as blurring, edge detection, sharpening, and projection operations.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_features_sec}

\begin{DoxyItemize}
\item Apply different types of 2D filters (\doxylink{class_blur}{Blur}, Edge Detection, Sharpening, etc.).
\item Perform 3D volume filtering using Gaussian and Median blur.
\item Extract 2D slices from 3D volumetric data.
\item Perform intensity projections (max, min, mean, median) on 3D volumes.
\item Convert images between different color spaces (RGB, HSV, HSL).
\end{DoxyItemize}
